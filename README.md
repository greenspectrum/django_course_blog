# Project "DJANGO blog"

## Description

Best blog on django.

## Installation

pip install -r requirements.txt

install mysql

## Run

cd app/blogengine

./manage.py runserver 8000


## Urls

http://127.0.0.1:8000/blog/tags/

http://127.0.0.1:8000/blog/tag/javascript/

http://127.0.0.1:8000/blog/tag/create/
